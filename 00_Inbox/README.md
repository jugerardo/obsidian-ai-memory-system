# 00_Inbox

Raw, untriaged captures land here. Drop exports from ChatGPT, Gemini, screenshots, voice memos, anything you haven't decided where to file yet.

## Goal

Keep the inbox **near zero** by triaging regularly. An inbox that fills up is a system that's failing.

## How it gets filled

- Imports from other AI tools
- "I'll sort this later" notes
- Voice notes / quick captures
- Emails or articles you want to process

## How it gets emptied

Move each item to its rightful home:

- One-off chat? → `10_Conversations/`
- Belongs to an active project? → `20_Projects/<project>/sessions/`
- Coding/debugging session with no project home? → `30_Code_Sessions/`
- Distilled idea worth keeping forever? → `40_Insights/`
- External doc/article? → `50_References/`

When you move a note, fill in its frontmatter with `status: raw` so the insight-extraction pass picks it up later.
