# 10_Conversations

Standalone AI conversations — single-session chats with no follow-up project home.

## When to file here vs. a project folder

- **Here**: a one-off question, a quick brainstorm, a "what is X" exploration that didn't turn into ongoing work.
- **In `20_Projects/<project>/sessions/`**: anything that's part of multi-session work toward a goal.

If a conversation here later sprouts into ongoing work, move it into the new project's `sessions/` folder.

## Subfolders

Organized by source, since exports from each tool tend to look different:

- `claude/` — sessions with Claude
- `chatgpt/` — exports from ChatGPT
- `gemini/` — exports from Gemini
- `other/` — anywhere else (Cursor, Copilot, local LLMs)

## Naming

`YYYY-MM-DD_short-slug.md` — e.g., `2026-04-28_obsidian-vault-design.md`

## Template

Use `_templates/conversation.md` (or let the wrap-up skill generate it for you).
