# 50_References

Durable external information — links, articles, cheat sheets, documentation excerpts. Stuff you didn't author but want to keep accessible.

## What belongs here

- Articles/blog posts you want to remember
- Documentation extracts you reference often
- Book notes, conference talk summaries
- Cheat sheets and lookup tables

## What does NOT belong here

- Your own ideas → `40_Insights/`
- Conversation transcripts → `10_Conversations/`

## Naming

`kebab-case-slug.md` — descriptive title.

## Frontmatter

```yaml
---
type: reference
date: 2026-04-28
source-url: https://example.com/article
tags: [topic1, topic2]
summary: "One-line description"
---
```
